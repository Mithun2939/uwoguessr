# UwoGuessr

A GeoGuessr-style game for Western University's campus. Guess where each photo was taken on an interactive map and compete on the leaderboard.

**Play:** [Mithun2939.github.io/uwoguessr](https://Mithun2939.github.io/uwoguessr)

## Features

- **Daily challenge** — 5 random campus locations each day  
- **Interactive map** — place your guess; points by distance  
- **Leaderboard** — daily, weekly, and all-time  
- **Contribute** — submit your own campus photos
